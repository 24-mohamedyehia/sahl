# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-02-01

### Added
- Initial release of Sahl
- Support for uploading new datasets to Kaggle
- Support for versioning existing datasets
- Semantic versioning support (MAJOR.MINOR.PATCH)
- Automatic metadata generation
- Version notes tracking
- Quick upload utility function
- Comprehensive documentation
- Modular code structure (prepare, upload, utils)

### Features
- `prepare_kaggle_dataset()` - Prepare datasets for upload
- `upload_to_kaggle()` - Upload with clear NEW/VERSION modes
- `quick_upload()` - One-liner upload function
- `print_upload_guide()` - Display usage guide
